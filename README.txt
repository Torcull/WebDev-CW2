To Run The Application:

1. Open 'WEBDEV-CW2' in VSCode or other similar IDE.
2. Make sure Node.js is installed, otherwise visit https://nodejs.org/en/download
4. Open terminal.
5. Enter 'npm install'
6. After previous command is complete, enter 'node index'
7. View application at localhost:3000 in your browser
